# Medical reference (normalization hints)

Use these as guidance when merging records; adapt to what appears in the source documents.

| Abbreviation / name | Typical meaning |
|---------------------|-----------------|
| WBC | White blood cell count |
| RBC | Red blood cell count |
| HGB / Hb | Hemoglobin |
| PLT | Platelet count |
| ALT / GPT | Alanine aminotransferase |
| AST / GOT | Aspartate aminotransferase |
| CREA / Cr | Creatinine |
| eGFR | Estimated glomerular filtration rate |
| UA / Uric acid | Uric acid |
| TC | Total cholesterol |
| TG | Triglycerides |
| LDL-C | LDL cholesterol |
| HDL-C | HDL cholesterol |
| FBG / GLU | Fasting blood glucose |
| HbA1c | Glycated hemoglobin |
| TSH | Thyroid-stimulating hormone |
| CRP | C-reactive protein |

## Pathology & immunohistochemistry (common markers)

| Abbreviation / name | Typical meaning |
|---------------------|-----------------|
| HE / H&E | Hematoxylin and eosin (routine histology stain) |
| IHC | Immunohistochemistry |
| Ki-67 | Proliferation index (often reported as % positive cells) |
| ER | Estrogen receptor |
| PR | Progesterone receptor |
| HER2 / neu | Human epidermal growth factor receptor 2 |
| PD-L1 | Programmed death-ligand 1 (often with TPS / CPS / IC score) |
| TPS | Tumor proportion score (PD-L1) |
| CPS | Combined positive score (PD-L1) |
| MMR / MSI | Mismatch repair proteins / microsatellite instability |
| MLH1, MSH2, MSH6, PMS2 | Mismatch repair proteins (Lynch syndrome screening) |
| p53 | Tumor protein p53 |
| EGFR | Epidermal growth factor receptor (mutation status in some cancers) |
| ALK, ROS1, BRAF, KRAS | Common oncogenic drivers (often mutation / rearrangement) |
| BRCA1 / BRCA2 | Breast cancer susceptibility genes |
| CK7, CK20, TTF-1, Napsin A | Lineage / site-of-origin markers (context-dependent) |
| CD3, CD4, CD8, CD20, CD45 | Lymphoid / immune markers |
| Gleason | Prostate cancer grade (score / pattern) |
| Nottingham | Breast cancer histologic grade (often G1–G3) |
| TNM | Tumor–node–metastasis stage |
| R0 / R1 / R2 | Resection margin status (no / microscopic / gross residual) |
| LVI | Lymphovascular invasion |
| PNI | Perineural invasion |

## Units

Keep units as written in the source when possible; do not convert unless the source explicitly provides converted values.

## Language

If the original record is in Chinese (or another language), preserve clinically relevant wording alongside normalized English test names when helpful.
