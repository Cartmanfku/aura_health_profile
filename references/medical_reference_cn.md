# 医学参考（标准化提示）

合并记录时可参考本表；以原始资料内容为准。

| MMR / MSI | 错配修复蛋白 / 微卫星不稳定 |

| p53 | p53 肿瘤抑制蛋白 |
| EGFR | 表皮生长因子受体（部分肿瘤关注突变状态） |

| BRCA1 / BRCA2 | 乳腺癌易感基因 |

| LVI | 淋巴血管侵犯 |
| PNI | 神经周围侵犯 |
